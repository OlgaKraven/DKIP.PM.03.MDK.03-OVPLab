Тема: Сниппеты и семантическая разметка

Теория Задание Контрольные вопросы

1. Теоретическая часть

1.1. Что такое сниппет

Сниппет — это фрагмент текста, отображаемый в результатах поисковой выдачи. Он обычно состоит из заголовка (<title>), URL и описания (meta description).

  • Формируется поисковой системой автоматически, но может быть частично задан в HTML.
  • Цель сниппета — привлечь внимание пользователя и побудить к клику.

1.2. Ключевые слова (ВЧ/СЧ/НЧ)

Ключевые слова — это поисковые запросы, по которым пользователи ищут информацию.

  • ВЧ (высокочастотные) — популярные, часто используемые запросы (например, «курсы программирования»).
  • СЧ (среднечастотные) — менее популярные, но более конкретные («курсы веб-разработки онлайн»).
  • НЧ (низкочастотные) — редкие, очень конкретные запросы («бесплатные курсы HTML и CSS 2025»).

1.3. Семантическая разметка

Семантические теги — это HTML-элементы, которые несут смысловую нагрузку и описывают структуру документа (например, <header>, <nav>, <main>, <article>, <section>, <footer>).

  • Помогают поисковым роботам лучше понимать содержание страницы.
  • Улучшают доступность для пользователей с ограниченными возможностями (скринридеры).
  • Упрощают поддержку и развитие кода.

2. Задание

Распределите ключевые слова по страницам сайта (например, index.html, about.html, contacts.html).

2.2. Составление сниппета (для index.html)

  • Создайте уникальный <title> (до 60 символов).
  • Создайте уникальное <meta name="description"> (до 160 символов).
  • Добавьте <meta property="og:title"> и <meta property="og:description"> для социальных сетей.

2.3. Применение семантической разметки

В HTML-файлах (index.html, about.html, contacts.html) замените или добавьте:

  • <header> для шапки сайта.
  • <nav> для навигационного меню.
  • <main> для основного контента.
  • <article> или <section> для логически завершённых частей.
  • <footer> для подвала.

2.4. Проверка

  1. Проверьте валидность HTML (https://validator.w3.org/).
  2. Проверьте сниппет в инструментах Google (Rich Results Test) или Яндекс (https://webmaster.yandex.ru/tools/snippet/).
  3. Убедитесь, что теги <title> и <meta name="description"> уникальны для каждой страницы.
