1. Теоретическая часть

1.1. Что такое SEO

SEO (Search Engine Optimization) — это комплекс мер по оптимизации сайта, направленных на повышение его видимости в результатах поисковых систем по релевантным запросам.

1.2. Основные этапы SEO

  • Краулинг — поисковый робот (бот) сканирует сайт, переходит по ссылкам.
  • Индексация — робот анализирует и сохраняет информацию о страницах в поисковом индексе.
  • Ранжирование — алгоритм определяет, какие страницы и в каком порядке показывать по конкретному запросу.

1.3. Основные факторы SEO

  • Технический SEO: скорость, безопасность (HTTPS), структура URL, robots.txt, sitemap.xml.
  • Контентный SEO: уникальность, релевантность, структура, ключевые слова.
  • Внешний SEO: ссылки с других сайтов (бэклинки).

1.4. Основные теги SEO

Тег/атрибут Пример Назначение
<title> <title>Курсы IT — Главная</title> Заголовок страницы в браузере и в сниппете поиска.
<meta name="description"> <meta name="description" content="Учебный сайт про курсы IT..."> Описание страницы в сниппете поиска.
<meta name="robots"> <meta name="robots" content="index, follow"> Инструкции для поисковых роботов (индексировать/не индексировать, переходить по ссылкам).
<h1> – <h6> <h1>Основной заголовок</h1> Структура контента. <h1> — обычно один на страницу.
<img src="..." alt="..."> <img src="cat.jpg" alt="Милый котик"> Описание изображения для поисковых систем и доступности.

2.3. Оптимизация about.html и contacts.html

  • Аналогично index.html, но с уникальными <title> и <meta name="description">.
  • Используйте заголовки <h1> – <h3> для структурирования текста.
  • Добавьте теги <img> с атрибутом alt.

2.4. Создание robots.txt и sitemap.xml

  • Создайте файл robots.txt в корне проекта.
  • Создайте файл sitemap.xml с URL всех трёх страниц.
  • Свяжите их.

2.5. Загрузка на GitHub Pages

  1. Создайте репозиторий на GitHub.
  2. Загрузите все файлы (index.html, about.html, contacts.html, robots.txt, sitemap.xml).
  3. Включите GitHub Pages (Settings → Pages).

2.6. Проверка и анализ

  1. Проверьте сайт в Google Search Console и Яндекс.Вебмастер.
  2. Проанализируйте сниппеты.
  3. Проверьте индексацию.
